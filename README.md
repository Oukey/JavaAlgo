### Связанный список
- представляет из себя цепочку узлов. Каждый узел хранит информацию (значение) и ссылку на следующий узел
- временная сложность:
  индексация    O(n)
  
  поиск O(n)
  
  вставка   O(1)
  
  удаление  O(1)
